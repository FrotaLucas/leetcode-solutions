Exercícios Fáceis (3)
1. Soma de subarray fixa

Dado um array de inteiros nums e um inteiro k, encontre a soma de todos os subarrays de tamanho k.

Exemplo:

nums = [1,2,3,4,5], k = 3
Resposta: [6, 9, 12]

💡 Dica: Use prefix sum para calcular cada janela em O(1).

2. Soma de intervalo (Prefix Sum básico)

Dado um array nums, responda várias queries do tipo:

qual a soma do índice L até R?

Exemplo:

nums = [2,4,6,8,10]
queries = [(1,3), (0,2)]
Resposta: [18, 12]

💡 Dica: Pré-calcule um array de prefix sum.

3. Contar frequência de números

Dado um array nums, retorne um dicionário com a contagem de cada número.

Exemplo:

nums = [1,2,2,3,1,1]
Resposta: {1:3, 2:2, 3:1}

💡 Dica: Use Dictionary direto.

🟡 Exercícios Médios (2)
4. Subarrays com soma igual a K

Dado um array nums e um inteiro k, conte quantos subarrays têm soma exatamente igual a k.

Exemplo:

nums = [1,1,1], k = 2
Resposta: 2

💡 Dica importante:
Use:

Prefix sum
Dictionary para armazenar quantas vezes uma soma apareceu

👉 Ideia:

se prefix_sum - k já apareceu → existe subarray válido
5. Subarray com soma zero (detectar existência)

Dado um array nums, verifique se existe algum subarray com soma 0.

Exemplo:

nums = [4,2,-3,1,6]
Resposta: True

💡 Dica:

Se duas prefix sums forem iguais → subarray entre elas soma 0
Use Dictionary ou Set
